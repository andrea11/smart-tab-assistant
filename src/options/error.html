Chrome AI API Support

As of December 1, 2024, Chrome already supports the AI API in the stable release. You'll need to download the latest Chrome and enable specific flags.

Once you've installed Chrome, open the browser and follow these steps:

  1. Go to chrome://flags/#prompt-api-for-gemini-nano

    Select "Enabled"

  2. Go to chrome://flags/#optimization-guide-on-device-model

    Select "Enabled BypassPrefRequirement"

  3. Go to chrome://components

    Click "Check for Update" on "Optimization Guide On Device Model" to download the model.
